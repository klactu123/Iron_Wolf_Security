Iron Wolf Threat Intel Brief Generator
======================================

Part of the Iron Wolf AI Security Toolkit.

Paste Indicators of Compromise (IOCs) — IP addresses, domains, file hashes,
URLs, email addresses, or CVE IDs — and get a structured threat intelligence
brief with attribution, MITRE ATT&CK mapping, risk assessment, and
recommended defensive actions.

The AI analyst uses live web search to research each IOC against current
threat intelligence sources.


QUICK START
-----------
1. Double-click "Start Threat Intel Brief.bat"
2. Open http://localhost:5176 in your browser
3. If first run, click Settings (gear icon) and enter your Anthropic API key
4. Paste IOCs and click "Generate Intel Brief"


MANUAL START
------------
1. Open a terminal in this folder
2. Run: npm install
3. Run: npm run dev
4. Open http://localhost:5176


PORTS
-----
- Frontend (Vite): 5176
- Backend (Express): 3004


API KEY
-------
Get an API key at https://console.anthropic.com
Enter it via the Settings modal or add to .env file:
  ANTHROPIC_API_KEY=sk-ant-your-key-here


SECURITY NOTES
--------------
- Server binds to localhost only (127.0.0.1)
- API key stored server-side only, never sent to browser
- Rate-limited endpoints prevent abuse
- All input validated and size-limited
- CORS restricted to configured origins
